4. 주석(comment)
(1) 소스 코드에 써 넣은 설명
(2) 컴파일 시에 사용안함
(3) 주석을 만드는 목적
    - 변수, 함수, 로직에 설명을 적기 위해
    - 프로그램 로직(퍼즐)이 복잡하여
      주석으로 플로우(흐름)을 미리 기술
      조금씩 프로그램으로 전환
      (Pseudo Coding : 수도 코딩)
    - 프로그램 시작 부분의 개요를 쓸 때
    - 이미 수행 테스트를 한 부분을 건너 뛰고 싶을 때
(4) 한 줄 주석, 구간 주석, 문서화 주석
    한 줄 주석 : //
    구간 주석 : /*  */
    문서화 주석 : /**    */


      // 1. 변수를 정의
