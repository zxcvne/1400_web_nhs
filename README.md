## 🗂️ Commit Convention

| Type | Description |
|------|-------------|
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `docs` | 문서 수정 (README 등) |
| `refactor` | 코드 리팩토링 |
| `style` | 코드 스타일 수정 (포맷, 세미콜론 등) |
| `chore` | 빌드, 설정, 기타 변경 |
