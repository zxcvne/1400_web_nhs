Day6

1. 반응형 레이아웃(Responsive Layout)
    - 웹의 레이아웃을 사용자의 해상도(가로)를 고려하여 디자인
    - 스마트폰, 태블릿 등은 화면이 작기 때문에 가독성이 중요
    - 이런 디자인 방법중 하나로 해상도에 따라 배치를 변경하는 레이아웃
    - @media를 통해 구현

2. 부트스트랩(BootStrap)
    - 세계에서 가장 인기있는 모바일 우선형/반응형
      Front-end 프레임워크
    - 프레임워크
        패키지 : 소스 + 디렉토리
        라이브러리 : 여러 개의 패키지
        프레임워크가 제공하는 공간에 라이브러리 탑제 가능
    - 가장 빈번하게 사용되는 CSS, Javascript를 미리 만들어놓고
      다운로드 하거나 CDN(Content Delivery Network)

(1) 설치 방법
    - 설치를 따로 하지는 않음
    - getbootstrap.com에서 다운로드 -> 압축해제
      또는 CSS인 경우 link 방식으로도 가능

(2) 탄생 배경
    - 트위터 프로젝트에서 시작
    - 서로 다른 인터페이스를 사용한 여러 개발자의 공동 작업
    - 디자인 불일치, 관리 난이도 상승, 코드의 방대함
      -> 일관성 있는 관리가 어려움
    - 트위터 개발자와 UI 디자이너가 최초로 개발

(3) 부트스트랩 
    - 프론트엔드 개발을 빠르고 쉽게 할 수 있는 프레임워크
    - HTML, CSS 기반의 템플릿 양식, 버튼, 내비게이션 등
    - 트위터에서 사용하는 각종 레이아웃, 버튼, 입력창 등
      디자인을 CSS와 JS로 구성
    - jQuery도 사용 가능
    - 상업적 이용 가능
    - 모바일/피씨/태블릿 디자인 지원
    - 검색을 통해 (free) BootStrap template으로 검색하면
      엄청난 양의 데이터가 검색됨
